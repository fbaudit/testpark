# 박재현 · 디지털 포렌식 전문가 — 프로필 사이트

완전 자립형 1페이지 사이트입니다. `index.html` 을 더블클릭하면 브라우저에서 바로 열립니다.
외부 라이브러리·인터넷 연결 없이 동작합니다.

## 실제 정보로 바꾸기

`index.html` 을 메모장/VS Code 로 열고, 아래 `[ ]` 로 표시된 부분을 본인 정보로 교체하세요.
`Ctrl+F` 로 `[` 를 찾으면 수정 지점이 순서대로 나옵니다.

| 위치 | 바꿀 내용 |
|------|-----------|
| Hero `hero-role` | 실제 직함·소속 |
| `.stats` 숫자 | 경력 연차, 사건 수 등 실제 수치 |
| About `about-card` | 프로필 요약 정보 |
| `#experience` 타임라인 | 실제 소속·기간·역할 |
| `#credentials` | 실제 보유 자격증·학력 (없는 항목은 삭제) |
| `#cases` | 공개 가능한 실제 실적 (익명·요약 권장) |
| `#contact` | 이메일·링크드인·연락처 (공개 가능한 것만) |

> ⚠️ 경력·실적 항목은 현재 **구성 예시**로 채워져 있습니다.
> 실제 이력과 다른 부분은 반드시 본인 정보로 교체하세요.

## 색상 테마 바꾸기

`index.html` 상단 `:root` 의 `--accent`, `--accent-2` 값을 바꾸면 전체 포인트 컬러가 바뀝니다.

- 원하시면 배포까지 도와드릴 수 있습니다.
